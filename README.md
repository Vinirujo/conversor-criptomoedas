# Conversor de Criptomoedas

Aplicação de terminal em Python para converter valores entre Bitcoin (BTC) e Real (R$), desenvolvida como atividade avaliativa na disciplina de programação da Universidade de Mogi das Cruzes (UMC).

## Funcionalidades

- Conversão de Bitcoin para Real
- Conversão de Real para Bitcoin
- Menu interativo em loop, com opção de sair

## Tecnologias

- Python 3

## Como executar

```bash
python conversor_de_criptomoeda.py
```

## O que aprendi

Praticar a criação de funções reutilizáveis, estruturação de menus com laços de repetição (`while`) e tratamento de entradas do usuário em Python.
